### ❌ 核心问题：动力学残差严重超标

**目标**: 动力学残差 < 0.1  
**实际**: 动力学残差 ≈ 83.25  
**差距**: **830倍**

#### 根本原因分析

**问题1: 欠驱动系统**
```
状态空间: 23-dim
  q = [q_base(4), θ_wheels(2), q_arm(6)]       = 12-dim
  v = [v_base(3), ω_wheels(2), v_arm(6)]       = 11-dim
  
广义加速度: 11-dim
  a = [a_base(3), α_wheels(2), a_arm(6)]
  
控制输入: 8-dim
  u = [τ_wheels(2), τ_arm(6)]
  
动力学方程: M*a + h = S^T*u
```

**关键矛盾**: 
- 11个广义加速度需要满足动力学约束
- 只有8个控制输入
- 基座的3个加速度 (a_base_x, a_base_y, a_base_yaw) **没有对应的直接扭矩控制**

**问题2: 选择矩阵S欠秩**
```python
S = [[0, 0, 0, 0, 0, 0, 0, 0],   # a_base_x  ← 无控制
     [0, 0, 0, 0, 0, 0, 0, 0],   # a_base_y  ← 无控制  
     [0, 0, 0, 0, 0, 0, 0, 0],   # a_base_yaw ← 无控制
     [1, 0, 0, 0, 0, 0, 0, 0],   # α_wheel_left
     [0, 1, 0, 0, 0, 0, 0, 0],   # α_wheel_right
     [0, 0, 1, 0, 0, 0, 0, 0],   # a_arm joints...
     ...]
```

前3行全为0，意味着基座动力学方程 **无法通过控制输入满足**。

**问题3: 动力学约束无法满足**
```
M*a + h = S^T*u

对于基座部分 (前3行):
  M[0:3, :] * a + h[0:3] = [0, 0, 0]  ← 因为S^T*u前3行为0
  
但是:
  ||h[0:3]|| ≈ 41.6 (重力+科氏力很大！)
  
结果:
  动力学残差 ≈ ||M*a + h - S^T*u|| ≈ 83.25
```

### 方案B: 修改WBC优化变量 (中等难度)

**核心思想**: WBC只优化有控制输入的自由度

**实施步骤**:

1. **重新定义优化变量**
```python
# 原始 (Phase 6当前):
z = [a(11), τ(8), λ_contact]  # a包含基座加速度

# 修改为 (方案B):
z = [α_wheels(2), a_arm(6), τ_wheels(2), τ_arm(6), λ_contact]
# 不优化基座加速度，由轮子运动学决定
```

2. **基座加速度从轮子推导**
```python
# 差速驱动运动学
v_base = J_diff @ ω_wheels  # (3,) = (3,2) @ (2,)
a_base = J_diff @ α_wheels  # 加速度同样通过雅可比

# 雅可比矩阵 (差速驱动几何关系)
J_diff = [
  [r/2,  r/2],     # v_x = r*(ωL + ωR)/2
  [0,    0],       # v_y = 0 (非完整约束)
  [r/L, -r/L]      # ω_yaw = r*(ωL - ωR)/L
]
```

3. **修改动力学约束**
```python
# 原始: M*a + h = S^T*τ (11个方程，8个控制)
# 修改: 只约束可控自由度

# 轮子和机械臂的动力学
M_controllable = M[3:11, 3:11]  # (8,8)
a_controllable = [α_wheels, a_arm]  # (8,)
h_controllable = h[3:11]  # (8,)

# QP约束:
M_controllable @ a_controllable + h_controllable = τ
```

4. **基座动力学作为软约束或忽略**
```python
# 基座部分不强制满足动力学 (欠驱动系统的现实)
# 或者添加为软约束:
cost += w_base_dynamics * ||M_base @ a_total + h_base||²
```

**预期效果**:
- 动力学残差: 从83.25降到 < 1.0
- 可控自由度完全满足动力学
- 基座运动由轮子运动学隐式保证

**工作量**: 2-3天
- 推导差速驱动雅可比: 0.5天
- 重构WBC代码: 1天
- 测试验证: 0.5-1天

### 方案C: 添加基座动力学耦合 (高难度)

**核心思想**: 完整建模基座-轮子动力学耦合

**实施步骤**:

1. **推导完整的轮式移动平台动力学**
```
包含:
- 基座质量和惯性
- 轮子惯性和摩擦
- 非完整约束 (vy_body = 0)
- 轮子-地面接触力
```

2. **构建完整的S矩阵**
```python
# 通过雅可比和动力学耦合填充S
S_full = construct_selection_matrix_with_coupling(
    robot_params, wheel_radius, wheelbase
)
# S_full[0:3, 0:2] 不再是0，而是包含轮子→基座的传递
```

3. **添加非完整约束**
```python
# Holonomic constraint: vy_body = 0
constraint: J_holonomic @ v = 0
# 作为QP的等式约束
```

**预期效果**:
- 动力学残差 < 0.1 (完全满足)
- 理论上最正确的建模
- 可能揭示新的控制问题

**工作量**: 5-7天
