# Phase 6-v2 使用指南

**推荐场景**: 单臂轮式 UR5e，位置执行器，高频平滑跟踪

---

## 架构

```
运动学 MPC (20 Hz, ALIGATOR)
    → TrajectoryInterpolator (25:1, 500 Hz)
    → FeedforwardPDController (u = u_mpc + Kp·e + Kd·ė)
    → MuJoCo position actuator
```

---

## 运行

```bash
export PYTHONPATH=.:../../build/bindings/python

pixi run -e all python scripts/demo_phase6_v2_simple.py
pixi run -e all python scripts/demo_phase6_v2.py          # 带 viewer
pixi run -e all python scripts/verify_phase6_v2_components.py
```

---

## 关键文件

| 文件 | 作用 |
|------|------|
| `phase6_controller.py` | 完整集成 |
| `trajectory_interpolator.py` | MPC 段 → 500 Hz |
| `feedforward_pd_controller.py` | 前馈 + PD |
| `aligator_mpc_controller.py` | 运动学 MPC 封装 |

模型: `assets/wheeled_ur5e.xml` 或 `assets/wheeled_dual_ur5e_v2.xml` (位置执行器版)

| 指标 | 值 |

| MPC 收敛率 | ~95–100% |
