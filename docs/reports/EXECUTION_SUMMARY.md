**核心问题**:
1. **积分器不匹配**: ALIGATOR (semi-implicit Euler) vs MuJoCo (implicitfast)
   - 单步误差: 7e-5 (可接受)
   - 25步累积: 0.034 (放大480倍！)

2. **求解器不收敛**: 50次迭代均未收敛

3. **"红球后退"现象**: 初始阶段误差突然增大

### Phase 4修复方案 (Phase 8计划)

**解决方案**: 实现implicit积分器以匹配MuJoCo

**预期修复后性能**:
- 单步误差: 7e-5 → 1e-6
- 25步累积: 0.034 → 0.001（减少34倍）
- 收敛率: 0% → 80%+
- EE误差: 2.5cm → < 2.0cm
