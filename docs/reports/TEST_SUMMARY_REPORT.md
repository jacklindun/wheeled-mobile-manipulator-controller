## 任务2: Phase 4模型修复测试

### Phase 4 当前状态 (2026-06-23)

**Phase 4架构**:
- 16维混合状态空间MPC
- 状态: x = [q_base(4), q_arm(6), v_arm(6)] = 16-dim
- 控制: u = [v_base(4), tau_arm(6)] = 10-dim
- 基座: 运动学积分（body-frame速度）
- 机械臂: ABA动力学 + semi-implicit Euler

**已知问题**:
1. **动力学模型与MuJoCo积分器不匹配**
   - 单步积分误差: 7e-5 (可接受)
   - 25步累积误差: 0.034 (480倍放大！)
   - 导致MPC预测与实际执行严重偏差

2. **求解器不收敛**
   - 收敛率: 0% (所有测试)
   - 迭代数: 50 (达到上限)
   - EE误差: 2.5-5.0 cm (次优解)

3. **"红球后退"现象**
   - t=0-1s: 红球突然向后移动，误差增大到5cm
   - t=1-3s: 红球重新追踪，误差降到2.5cm
   - t>3s: 稳定在2.5cm误差（次优解）
